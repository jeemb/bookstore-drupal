# _Books and Stuff_

#### Drupal Website 4.21.2017

#### By Jennifer Beem

## Description

This Drupal-built webpage is for a small bookstore. It organizes book reviews and holds basic information for the bookstore. 

## Setup/Installation Requirements

* Clone this repository
* Open MAMP and set server root to project root
* Import database stored in sites/db-backup into MySQL
* Navigate into document root via computer terminal
* Open up web browser and go to `localhost:8888â€™
